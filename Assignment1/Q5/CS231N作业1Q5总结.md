## 三、实验核心流程与详细步骤
### 模块一：全连接神经网络实现与正确性验证
#### 1. 网络结构设计（`cs231n/classifiers/fc_net.py`）
- **核心结构**：输入层 → 多层（Affine + ReLU） → 输出层（Affine + Softmax）
- **可配置参数**：隐藏层层数/神经元数、输入维度、类别数、L2正则化强度、权重初始化尺度、数据类型。
- **模块化复用**：调用`cs231n/layers.py`中已实现的`affine_forward/backward`、`relu_forward/backward`、`softmax_loss`，降低代码冗余。

#### 2. 前向传播实现
```python
# 核心逻辑（简化版）
def forward(self, X):
    cache = []
    out = X.reshape(X.shape[0], -1)  # 展平输入（3×32×32→3072维）
    for i in range(len(self.params)//2):
        W, b = self.params[f'W{i+1}'], self.params[f'b{i+1}']
        out, aff_cache = affine_forward(out, W, b)
        out, relu_cache = relu_forward(out)
        cache.append((aff_cache, relu_cache))
    # 输出层（无ReLU）
    W_out, b_out = self.params[f'W{len(self.params)//2+1}'], self.params[f'b{len(self.params)//2+1}']
    out, aff_cache = affine_forward(out, W_out, b_out)
    cache.append(aff_cache)
    return out, cache
```

#### 3. 反向传播与损失计算
```python
def loss(self, X, y=None):
    out, cache = self.forward(X)
    if y is None:
        return out
    # 计算Softmax交叉熵损失 + L2正则化
    data_loss, dscores = softmax_loss(out, y)
    reg_loss = 0.5 * self.reg * sum(np.sum(W**2) for W in self.params.values() if 'W' in W)
    loss = data_loss + reg_loss
    
    # 反向传播计算梯度
    grads = {}
    # 输出层梯度
    W_out = self.params[f'W{len(self.params)//2+1}']
    d_out, grads[f'W{len(self.params)//2+1}'], grads[f'b{len(self.params)//2+1}'] = affine_backward(dscores, cache[-1])
    grads[f'W{len(self.params)//2+1}'] += self.reg * W_out  # 正则化梯度
    
    # 隐藏层梯度（反向遍历）
    for i in reversed(range(len(self.params)//2)):
        aff_cache, relu_cache = cache[i]
        d_out = relu_backward(d_out, relu_cache)
        d_out, grads[f'W{i+1}'], grads[f'b{i+1}'] = affine_backward(d_out, aff_cache)
        grads[f'W{i+1}'] += self.reg * self.params[f'W{i+1}']  # 正则化梯度
    return loss, grads
```

#### 4. 梯度校验（关键正确性验证步骤）
- **校验原理**：通过有限差分法计算**数值梯度**，与反向传播得到的**解析梯度**对比，验证反向传播代码无Bug。
- **校验流程**：
  1. 生成小批量随机输入`X`（2个样本，15维）和标签`y`；
  2. 分别在正则化强度`reg=0`和`reg=3.14`下，计算模型损失与解析梯度；
  3. 对每个参数（`W1/W2/W3/b1/b2/b3`），用`eval_numerical_gradient`计算数值梯度；
  4. 计算相对误差：`rel_error = |数值梯度 - 解析梯度| / (|数值梯度| + |解析梯度|)`。
- **合格标准**：相对误差≤1e-7（少数参数如`W2`在`reg=0`时可放宽至1e-5）。
- **实验结果**：所有参数相对误差均满足要求，验证反向传播实现正确。

#### 5. 初始损失验证
- 随机初始化权重（`weight_scale=5e-2`），未训练时的初始损失应接近`log(10)≈2.3`（10类分类的随机猜测损失）。
- 实验结果：`reg=0`时初始损失2.300，`reg=3.14`时初始损失7.052（正则化增加了损失），符合预期。

### 模块二：浅层与深层网络训练难度对比
#### 1. 实验设计
- **目标**：验证网络深度对训练敏感性的影响，通过小样本（50张训练图）过拟合测试。
- **两组对比**：
  - 3层网络：输入层→100神经元→100神经元→输出层；
  - 5层网络：输入层→100神经元×4→输出层。
- **核心变量**：仅调整`weight_scale`（权重初始化尺度）和`learning_rate`（学习率），其他参数一致（`batch_size=25`、`num_epochs=20`、`SGD优化`）。

#### 2. 实验过程与现象
- **3层网络**：`weight_scale=3e-2`、`learning_rate=2e-3`，训练10轮后训练损失降至接近0，训练准确率100%，轻松过拟合。
- **5层网络**：初始尝试`weight_scale=1e-2`、`learning_rate=5e-3`时，损失震荡不下降（梯度爆炸）；调整为`weight_scale=5e-2`、`learning_rate=1e-2`后，训练20轮达成100%训练准确率，但收敛速度比3层网络慢。

#### 3. 关键结论
- 深层网络对`weight_scale`和`learning_rate`更敏感：权重过小导致梯度消失，过大导致梯度爆炸；学习率需精准匹配，否则无法收敛。
- 根本原因：深层网络的梯度传播经过更多层，初始权重的微小偏差会被放大，导致梯度消失/爆炸。

### 模块三：优化器实现与性能对比
#### 1. 四类优化器原理与实现（`cs231n/optim.py`）
| 优化器          | 核心原理                                                                 | 关键公式（简化）                                                                 |
|-----------------|--------------------------------------------------------------------------|----------------------------------------------------------------------------------|
| 原始SGD         | 固定学习率，统一更新所有参数                                             | `w = w - lr * dw`                                                                |
| SGD+Momentum    | 引入速度变量`v`，累积历史梯度，模拟“惯性”                                 | `v = mu * v - lr * dw`；`w = w + v`                                              |
| RMSProp         | 指数滑动平均累积梯度平方，逐参数自适应学习率，避免AdaGrad的学习率衰减     | `cache = decay_rate * cache + (1-decay_rate) * dw²`；`w = w - lr * dw / sqrt(cache+eps)` |
| Adam            | 融合Momentum（一阶矩）和RMSProp（二阶矩），加入偏差修正                   | `m = beta1*m + (1-beta1)*dw`；`v = beta2*v + (1-beta2)*dw²`；`w = w - lr * m_hat / sqrt(v_hat+eps)` |

#### 2. 优化器正确性验证
- 对每个优化器，输入预设的`w`、`dw`、初始缓存（`v/m/cache`），验证输出的`next_w`和更新后的缓存与预期值一致，相对误差≤1e-7。
- 示例结果：Adam优化器`next_w`误差1.14e-7，`m`/`v`误差4.21e-9，验证实现正确。

#### 3. 优化器性能对比实验
- **实验设计**：6层全连接网络（100神经元×5），训练集4000张样本，`num_epochs=5`、`batch_size=100`，对比四类优化器的收敛速度与精度。
- **关键参数**：
  - SGD：`lr=5e-3`；
  - SGD+Momentum：`lr=5e-3`、`mu=0.9`；
  - RMSProp：`lr=1e-4`、`decay_rate=0.99`；
  - Adam：`lr=1e-3`、`beta1=0.9`、`beta2=0.999`。

#### 4. 实验结果与分析
| 优化器          | 训练损失下降速度 | 5轮后训练准确率 | 5轮后验证准确率 | 核心优势                     |
|-----------------|------------------|------------------|------------------|------------------------------|
| 原始SGD         | 最慢             | 45%              | 43%              | 简单易实现，无额外超参数     |
| SGD+Momentum    | 较快             | 58%              | 55%              | 收敛速度优于SGD，缓解震荡    |
| RMSProp         | 快               | 62%              | 57%              | 自适应学习率，适合深层网络   |
| Adam            | 最快             | 65%              | 59%              | 收敛最快，稳定性最强，泛化好 |

- **关键结论**：Adam融合了前三者的优势，是深层网络训练的最优选择；原始SGD在深层网络中收敛极慢，不推荐使用。

### 模块四：最优模型训练与泛化能力评估
#### 1. 超参数调优（核心影响因素）
通过多组实验迭代，确定最优超参数组合：
| 超参数                | 取值          | 选择依据                                                                 |
|-----------------------|---------------|--------------------------------------------------------------------------|
| 隐藏层结构            | [512, 256, 128] | 深度适中（3层隐藏层），避免梯度消失；神经元数量递减，逐步压缩特征维度     |
| 权重初始化尺度        | 2e-2          | 兼顾梯度传播稳定性，避免初始梯度过大/过小                                 |
| 学习率                | 5e-4          | Adam优化器适配的中等学习率，避免后期震荡                                 |
| L2正则化强度          | 1e-4          | 轻微正则化，抑制过拟合同时不影响训练精度                                 |
| 训练轮数（num_epochs）| 20            | 15轮后验证精度趋于稳定，继续训练无提升                                   |
| 批量大小（batch_size） | 128           | 平衡训练速度与梯度估计准确性                                             |
| 优化器                | Adam          | 收敛速度最快，稳定性最优                                                 |

#### 2. 训练过程监控
- **损失曲线**：训练损失从初始2.85逐步下降至0.7左右，无明显震荡，说明收敛稳定。
- **精度曲线**：训练准确率从16.5%提升至79.4%，验证准确率最高达54.7%（第17轮），后期无明显过拟合（训练/验证精度差距约25%）。

#### 3. 泛化能力评估
- **验证集准确率**：54.7%（达标，≥50%）；
- **测试集准确率**：53%（与验证集差距小，泛化能力良好）；
- **结果分析**：全连接网络仅能利用像素的全局特征，无法捕捉图像的空间局部特征，导致精度存在上限（CIFAR-10上全连接网络极限精度约55%-60%）。

## 五、实验问题与解决方案
### 1. 遇到的核心问题
| 问题描述                                  | 原因分析                                                                 | 解决方案                                                                 |
|-------------------------------------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------|
| 梯度校验时`W2`相对误差达1e-4（超1e-7）    | 反向传播中正则化梯度未正确添加                                           | 检查`affine_backward`后是否加入`grads[W] += reg * W`                     |
| 深层网络训练时损失震荡不下降              | 权重初始化尺度过大，导致梯度爆炸                                         | 减小`weight_scale`（从1e-1调整为2e-2），降低学习率（从1e-3调整为5e-4）   |
| Adam优化器训练后期精度震荡                | 学习率过高，导致参数在最优值附近震荡                                     | 降低学习率（从1e-3调整为5e-4），延长训练轮数                             |
| 验证集精度始终低于50%                     | 隐藏层结构过浅（仅1层256神经元），特征提取能力不足                       | 加深网络结构（调整为[512,256,128]），提升模型容量                       |

### 2. 关键经验总结
- 梯度校验是反向传播代码调试的“金标准”，必须在训练前完成，避免后续训练无效；
- 超参数调优需“单一变量法”：每次仅调整一个超参数，观察其对结果的影响，避免多变量干扰；
- 深层网络优先选择Adam/RMSProp优化器，原始SGD难以收敛；
- 正则化强度不能过大：过强的正则化会导致模型欠拟合，训练精度和验证精度均偏低。

## 六、实验总结
本次实验完整实现了多层全连接神经网络的设计、训练与优化，达成了预期目标（测试集准确率53%）。通过实验，系统掌握了深度网络的训练原理、优化策略及工程实践技巧，理解了网络深度、初始化方式、优化器选择、超参数调优等因素对模型性能的关键影响。

从学习者角度，本次实验不仅巩固了深度学习的理论基础，更培养了问题排查、实验设计、结果分析的工程能力。同时，也认识到全连接网络的局限性——无法利用图像的空间特征，为后续学习卷积神经网络（CNN）、提升图像分类精度奠定了基础。

实验最终证明：合理的网络结构设计、适配的优化器、精准的超参数调优，是深度全连接网络训练成功的核心三要素。



# Assignment2 Q1作业总结

## 一、Batch Normalization（BN 批量归一化）核心基础知识点

### 1. 提出动机：内部协变量偏移 Internal Covariate Shift
1. 神经网络训练时，**每一层输入分布会随前面权重更新不断变化**；
2. 即使输入数据做了归一化，深层网络的激活值均值、方差依然会漂移；
3. 分布漂移导致训练变慢、对初始化敏感、学习率不能设太大、深层网络难以收敛；
4. BN 目的：**固定每一层激活值的分布，减少内部协变量偏移，加速训练、提升稳定性**。

### 2. BN 核心思想
对**同一个批次（batch）**内，**每个特征维度**做：
均值归零 → 方差归一 → 再用可学习参数 **缩放γ、偏移β** 恢复表达能力。
> 不是强行限制分布为标准正态，而是让网络自己学最优分布。

### 3. BN 前向传播公式（训练阶段）
设输入：$X\in \mathbb{R}^{N\times D}$，$N$样本数，$D$特征维度
1. 批次均值：
$$\mu = \frac1N\sum_{i=1}^N x_i$$
2. 批次方差：
$$\sigma^2 = \frac1N\sum_{i=1}^N (x_i-\mu)^2$$
3. 归一化：
$$\hat x_i = \frac{x_i - \mu}{\sqrt{\sigma^2+\epsilon}}$$
4. 缩放偏移（可学习）：
$$y_i = \gamma \hat x_i + \beta$$

### 4. 训练 & 测试阶段差异
- **训练时**：用**当前 mini-batch** 计算均值、方差；同时维护**全局滑动均值/方差**（running mean/var）。
- **测试时**：**不用当前 batch 统计量**，直接用训练累积的滑动均值和方差做归一化。
- 原因：测试时 batch 可以很小甚至单样本，统计量无意义。

### 5. BN 反向传播两种实现
1. **计算图逐节点求导**：拆解每一步运算，逐层反向传梯度，直观但冗余、速度慢；
2. **数学公式化简版（alt 版本）**：直接推导出 $dx,d\gamma,d\beta$ 闭式解，**精度一致、速度更快**（实验中加速近2倍）。

### 6. BN 放置位置
标准位置：**全连接/卷积层 → BN → ReLU**
> 本实验要求：每个非线性激活前插入 BN，最后输出层不加 BN。

## 二、BN 三大实验结论 & 底层原理
### 实验1：BN 与 权重初始化尺度
#### 实验现象
1. **无 BN 网络**
   - 对权重初始化**极度敏感**；
   - 初始化太小：梯度消失，完全无法训练；
   - 只有极窄区间能收敛，性能上限低；
   - 初始化过大：训练震荡、不收敛。
2. **有 BN 网络**
   - 对初始化尺度**极鲁棒**；
   - 从极小到大范围初始化都能正常收敛、保持高准确率；
   - 不需要精细调初始化“甜蜜点”。

#### 原理
BN 把**层激活值的分布和权重初始化规模解耦**，无论权重初始大小，都强制把每层激活拉到稳定分布，杜绝梯度消失/爆炸。

---

### 实验2：BN 与 Batch Size 批次大小
#### 实验现象
1. BN 性能**强依赖 batch size**：
   - batch 越大：均值方差估计越准，BN 效果越好、收敛越快、泛化更稳；
   - batch 很小（如=5）：批次统计量噪声极大，归一化失效，BN 优势消失，甚至不如无BN。
2. 无BN小batch 本身收敛就慢、精度低。

#### 原理
BN 依赖**批次样本估计整体分布**，小批次无法代表真实数据分布，均值方差估计偏差大，反而破坏特征分布。

## 三、Layer Normalization（LN 层归一化）知识点
### 1. 提出背景
BN 致命缺陷：**依赖 batch size**，小批量、RNN、Transformer 等场景没法用；LN 为解决这一问题提出。

### 2. LN 与 BN 归一化维度本质区别
| 归一化 | 归一化维度 | 统计对象 | 适用场景 |
|--------|------------|----------|----------|
| BN | 按**列**（同一特征、跨样本） | 同一个batch里所有样本的同一维度 | CNN、大批量训练 |
| LN | 按**行**（同一样本、跨特征） | 单一样本自身所有特征维度 | 小batch、RNN、Transformer、NLP |

### 3. LN 核心特点
1. 对**每个单独样本**，在自身所有特征上做均值方差归一化；
2. **没有滑动均值/方差**，**训练和测试流程完全一样**；
3. 完全**不依赖 batch size**，小批量也能用；
4. 同样带可学习 $\gamma,\beta$ 缩放偏移参数。

### 4. LN 实验结论
1. 改变 batch size 对 LN 训练精度、收敛速度**影响极小**；
2. 相比 BN，LN 彻底摆脱了批次大小限制；
3. 缺点：特征维度很小时，自身统计量噪声大，LN 效果会变差。

## 四、四道随堂问答题核心知识点（对应实验里 Inline Question）
### Q1：BN+初始化 问答总结
BN 平滑了权重初始化的影响，无BN深度网络必须精调初始化，BN网络天然鲁棒，因为归一化稳定了每层激活分布。

### Q2：BN+Batch Size 问答总结
BN 效果随 batch 增大而变好，小batch 统计不准导致BN失效；根源是BN基于批次统计，LN无此问题。

### Q3：预处理操作类比 BN/LN
- **类比BN**：全局数据集级均值归一化（操作3），跨样本统计；
- **类比LN**：单张图像内部所有特征归一化（操作2），单样本内统计；
- 按行像素求和、阈值二值化 都不属于二者。

### Q4：LN 什么时候效果差
**特征维度 very small 时**效果差。
原因：LN 靠单样本自身所有特征算均值方差，特征维度太少，统计量噪声极大，归一化失去意义；
深层网络、正则大不是LN失效的核心原因。

---

## 五、BN vs LN 对比总结
1. **维度**：BN 跨样本归一化；LN 跨特征归一化
2. **Batch 依赖**：BN 强依赖；LN 完全不依赖
3. **训练测试**：BN 训测不同；LN 训测一致
4. **初始化敏感**：BN 降低初始化敏感；LN 也鲁棒，但原理不同
5. **适用场景**：BN 适合CNN、大批量；LN 适合RNN/Transformer、小批量
6. **参数**：都有可学习 $\gamma,\beta$，保留网络表达能力

## 六、代码实现层面知识点
1. BN/LN 前向：计算均值→方差→归一化→缩放偏移，保存中间变量给反向传播；
2. BN 反向有两种写法：计算图版、化简公式版，结果一致但速度不同；
3. 全连接网络改造：增加 normalization 分支，支持 None / batchnorm / layernorm；
4. 梯度检查：用数值梯度验证解析梯度，误差在 $10^{-8}\sim10^{-13}$ 说明实现正确；
5. 实验框架：Solver 封装训练逻辑，可对比不同归一化、初始化、batch size 的收敛曲线。
